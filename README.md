# AlorianLauncher
AlorianMC Minecraft Başlatıcısı (AlorianLauncher), AlorianMC sunucusunun güvenilir minecraft başlatıcı uygulamasıdır.
Kolay bir şekilde uygulamayı kullanarak 1.16.5 sürümünde Minecraft oynayabilirsiniz.
